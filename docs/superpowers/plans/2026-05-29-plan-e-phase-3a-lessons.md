# Plan E — Phase 3a (Phoenix lessons 21–25) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Author the five pre-database Phoenix lessons (21 Plug standalone; 22–25 growing the in-memory `Tracker` app to a projects index + new-project form behind a context).

**Architecture:** Lesson 21 is a standalone Plug Mix project. Lessons 22–25 carry the threaded `Tracker` app: each lesson's `exercises/` = the previous lesson's `solutions/` plus new `@tag :pending` tests and new stubs; `solutions/` implements them. The app is generated by `mix phx.new tracker` with the Ecto **Repo kept dormant** (removed from the supervision tree; `test` alias and sandbox setup stripped) so no Postgres is needed. Projects live in an `Agent`-backed `Tracker.ProjectStore` behind a `Tracker.Projects` context.

**Tech Stack:** Elixir 1.18 (CI pin) / OTP 27; Phoenix `~> 1.8` (1.8.7); `phoenix_live_view ~> 1.1`; Plug `~> 1.16`; ExUnit; reveal.js slides. Generated with the `phx_new` 1.8.5 installer.

**Spec:** `docs/superpowers/specs/2026-05-28-phase-3a-design.md`.

### Threading procedure (lessons 23, 24, 25)

Each of these lessons starts from the previous lesson's **solutions/**:

1. `tools/new-lesson NN-slug` to create the lesson dir (gives README, HINTS,
   `slides/index.html`, `slides/slides.md` with correct reveal paths, and
   template `exercises/`/`solutions/`).
2. `rm -rf lessons/NN-slug/exercises lessons/NN-slug/solutions`.
3. `cp -R lessons/PREV/solutions lessons/NN-slug/exercises` and
   `cp -R lessons/PREV/solutions lessons/NN-slug/solutions`.
4. In **both** new dirs: strip `@tag :pending` from the previous lesson's
   carried tests (they are now passing baseline). Keep `exercises/`'s
   `test_helper.exs` excluding pending and `solutions/`'s not excluding.
5. Add this lesson's new test file(s) with `@tag :pending` to **both** dirs
   (identical files).
6. In `exercises/`, replace this lesson's new code with `raise "TODO: ..."`
   stubs. In `solutions/`, write the real implementation.
7. Run `mix format` in both dirs before committing.

This satisfies both CI invariants: `exercises/` passes `mix test` (pending
excluded → only carried baseline tests run, and they pass); `solutions/`
passes `mix test --include pending`.

```elixir
defmodule AuthPlug do
  @moduledoc ~s|A module plug: halts with 401 unless the "x-token" header is "secret".|
  import Plug.Conn

  def init(opts), do: opts

  def call(conn, _opts) do
    case get_req_header(conn, "x-token") do
      ["secret"] -> conn
      _ -> conn |> send_resp(401, "unauthorized") |> halt()
    end
  end
end
```

`api_router.ex`:

```elixir
defmodule ApiRouter do
  @moduledoc "A Plug.Router composing the plugs into a pipeline."
  use Plug.Router

  plug :match
  plug :dispatch

  get "/hello", do: send_resp(conn, 200, "hello")

  get "/secret" do
    conn = AuthPlug.call(conn, AuthPlug.init([]))
    if conn.halted, do: conn, else: send_resp(conn, 200, "top secret")
  end
end
```

```elixir
  @moduledoc "In-memory project storage backed by an Agent. Replaced by Postgres in lesson 29."
  use Agent

  def start_link(_opts), do: Agent.start_link(fn -> %{items: [], next_id: 1} end, name: __MODULE__)

  def add(attrs) do
    Agent.get_and_update(__MODULE__, fn %{items: items, next_id: id} ->
      project = Map.put(attrs, :id, id)
      {project, %{items: [project | items], next_id: id + 1}}
```

```elixir
  use TrackerWeb, :controller

  alias Tracker.ProjectStore

  @types %{name: :string, status: :string}

  def index(conn, _params), do: render(conn, :index, projects: ProjectStore.list())

  def new(conn, _params) do
    render(conn, :new, form: Phoenix.Component.to_form(change(), as: :project))
  end

  def create(conn, %{"project" => params}) do
    changeset = change(params)

    if changeset.valid? do
      changeset |> Ecto.Changeset.apply_changes() |> ProjectStore.add()

      conn
      |> put_flash(:info, "Project created.")
      |> redirect(to: ~p"/projects")
    else
      render(conn, :new, form: Phoenix.Component.to_form(%{changeset | action: :insert}, as: :project))
    end
  end

  defp change(attrs \\ %{}) do
    {%{status: "open"}, @types}
    |> Ecto.Changeset.cast(attrs, Map.keys(@types))
    |> Ecto.Changeset.validate_required([:name])
  end
end
```
